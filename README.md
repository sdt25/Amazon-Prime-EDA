# Amazon Prime Exploratory Data Analysis (EDA)

## Project Overview
This project focuses on performing exploratory data analysis on Amazon Prime content data to understand trends, content distribution, and platform insights.

## Objectives
- Analyze distribution of movies and TV shows
- Study content trends across years
- Identify popular genres and ratings
- Generate insights useful for business and content strategy decisions

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Insights
- Amazon Prime content has increased significantly over recent years
- Movies dominate the platform compared to TV shows
- Certain genres and ratings are more prevalent
- Data-driven insights can help guide content acquisition strategies
